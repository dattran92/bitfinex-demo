# Bitfinex Demo

## About the project

A very basic replication on some components on bitfinex website.

## View Demo
[https://dattran92.github.io/bitfinex-demo/](https://dattran92.github.io/bitfinex-demo/)

## What is used in
- react
- react-redux
- redux-thunk
- websocket

## Script

### Install dependencies
```
yarn

```
### Run development
```
yarn run dev
```
### Build production
```
yarn build
```
### Deploy to github pages
```
yarn run deploy
```
